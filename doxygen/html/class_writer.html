Detailed Description

Class providing an interface to write a Variant object into a JSON syntax.

The content of a Variant object is wrote in an output stream with a JSON like syntax.

See also
Variant, Writer

Constructor & Destructor Documentation

Writer::Writer ( std::ostream *  output)

Construct a Writer object with the specified output stream.

The Writer object will use the output stream to write JSON data. The operation can be performed using the write method.

Note
The stream is supposed to be ready to write. Otherwise, an exception can be thrown.
