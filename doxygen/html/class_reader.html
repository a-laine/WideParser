Detailed Description

Class providing an interface to read a JSON input.

Parsing methods construct a Variant object containing the data of the stream in a similar structure.

See also
Variant, Writer

Constructor & Destructor Documentation

Reader::Reader ( std::istream *  input)

Construct a Reader object with the specified input stream.

The Reader object will use the input stream to extract JSON data. The parsing operation can be performed using the parse method.

Note
The stream is supposed to be ready to read. Otherwise, an exception can be thrown.
Parameters
inputThe input stream to read.
Exceptions
std::logic_erroris thrown if the stream is not good.
